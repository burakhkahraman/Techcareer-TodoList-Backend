# ToDo App Spring Boot Project


## Requirements
- Java 17
- Node.js
- IntelliJ IDEA
- Visual Studio Code


## Technology

### Backend
* Javase
* Jdbc CRUD Native
* Spring Core
* Spring MVC
* Spring Rest

### Frontend
- HTML5
- CSS3
- Responsive Design
- JS
- React

### Database
- *PostgreSQL*

Frontend

- 2- npm install axios
- 3- npm start
